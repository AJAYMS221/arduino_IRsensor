# arduino_IRsensor
An IR (Infrared) sensor is an electronic device that detects infrared radiation (IR) emitted by objects. It is widely used in automation, security systems, line-following robots, proximity sensing, and many other applications. IR sensors are capable of both emitting and detecting infrared light.
